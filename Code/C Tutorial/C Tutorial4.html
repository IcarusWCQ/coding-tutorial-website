If Statements

  1. Write a program that checks if a number is positive. Declare an integer variable num, assign it a value, and print "The number is positive" if the number is greater than 0 using an if statement.
    2.                     
#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("The number is positive\n");
    }

    return 0;
}
                    
                
                        Sample Output:
                    
The number is positive
  2. Write a program that checks if a number is even or odd. Declare an integer variable num, assign it a value, and print "The number is even" if the number is divisible by 2 using an if statement.
    3.                     
#include <stdio.h>

int main() {
    int num = 4;

    if (num % 2 == 0) {
        printf("The number is even\n");
    } else {
        printf("The number is odd\n");
    }

    return 0;
}
                    
                
                        Sample Output:
                    
The number is even
  3. Write a program that checks if a character is a vowel. Declare a char variable ch, assign it a value, and print "The character is a vowel" if the character is 'a', 'e', 'i', 'o', or 'u' using an if statement.
    4.                     
#include <stdio.h>

int main() {
    char ch = 'a';

    if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u') {
        printf("The character is a vowel\n");
    } else {
        printf("The character is not a vowel\n");
    }

    return 0;
}
                    
                
                        Sample Output:
                    
The character is a vowel
  4. Write a program that checks if a number is negative, zero, or positive. Declare an integer variable num, assign it a value, and print "The number is negative", "The number is zero", or "The number is positive" using an if statement.
    5.                     
#include <stdio.h>

int main() {
    int num = -5;

    if (num < 0) {
        printf("The number is negative\n");
    } else if (num == 0) {
        printf("The number is zero\n");
    } else {
        printf("The number is positive\n");
    }

    return 0;
}
                    
                
                        Sample Output:
                    
The number is negative
  5. Write a program that checks if a year is a leap year. Declare an integer variable year, assign it a value, and print "The year is a leap year" if the year is divisible by 4 but not 100, or if it is divisible by 400 using an if statement.
    6.                     
#include <stdio.h>

int main() {
    int year = 2000;

    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        printf("The year is a leap year\n");
    } else {
        printf("The year is not a leap year\n");
    }

    return 0;
}
                    
                
                        Sample Output:
                    
The year is a leap year
