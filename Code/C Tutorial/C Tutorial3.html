Data Types

  1. Declare an integer variable `age`, a float variable `height`, and a char variable `initial`. Assign them values and print them using `printf`.
    2.                     
#include <stdio.h>

int main() {
    int age = 21;
    float height = 5.8;
    char initial = 'J';
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Initial: %c\n", initial);
    return 0;
}
                    
                
                        Sample Output
                    
Age: 21
Height: 5.8
Initial: J
  2. Declare a double variable `distance`, assign it a value, and print it using `printf`.
    3.                     
#include <stdio.h>

int main() {
    double distance = 12345.6789;
    printf("Distance: %.4lf\n", distance);
    return 0;
}
                    
                
                        Sample Output
                    
Distance: 12345.6789
