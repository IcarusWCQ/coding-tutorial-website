Switch Case

Day of the Week

  1. Write a Java program that takes an integer input (1-7) and prints the corresponding day of the week using a switch case. 
                            Sample Output
                        
                            Enter a number (1-7): 3
                            Wednesday
                        
                    
                                
                                import java.util.Scanner;

                                public class DayOfWeek {
                                    public static void main(String[] args) {
                                        Scanner scanner = new Scanner(System.in);
                                        System.out.print("Enter a number (1-7): ");
                                        int day = scanner.nextInt();
                                        switch (day) {
                                            case 1:
                                                System.out.println("Sunday");
                                                break;
                                            case 2:
                                                System.out.println("Monday");
                                                break;
                                            case 3:
                                                System.out.println("Tuesday");
                                                break;
                                            case 4:
                                                System.out.println("Wednesday");
                                                break;
                                            case 5:
                                                System.out.println("Thursday");
                                                break;
                                            case 6:
                                                System.out.println("Friday");
                                                break;
                                            case 7:
                                                System.out.println("Saturday");
                                                break;
                                            default:
                                                System.out.println("Invalid input");
                                                break;
                                        }
                                    }
                                }

    2. Calculator

  2. Write a Java program that performs basic arithmetic operations (addition, subtraction, multiplication, division) based on user input using a switch case. 
                            Sample Output
                        
                            Enter first number: 10
                            Enter second number: 5
                            Enter operation (+, -, *, /): *
                            Result: 50
                        
                    
                                
                                import java.util.Scanner;

                                public class Calculator {
                                    public static void main(String[] args) {
                                        Scanner scanner = new Scanner(System.in);
                                        System.out.print("Enter first number: ");
                                        int num1 = scanner.nextInt();
                                        System.out.print("Enter second number: ");
                                        int num2 = scanner.nextInt();
                                        System.out.print("Enter operation (+, -, *, /): ");
                                        char operation = scanner.next().charAt(0);

                                        switch (operation) {
                                            case '+':
                                                System.out.println("Result: " + (num1 + num2));
                                                break;
                                            case '-':
                                                System.out.println("Result: " + (num1 - num2));
                                                break;
                                            case '*':
                                                System.out.println("Result: " + (num1 * num2));
                                                break;
                                            case '/':
                                                if (num2 != 0) {
                                                    System.out.println("Result: " + (num1 / num2));
                                                } else {
                                                    System.out.println("Cannot divide by zero");
                                                }
                                                break;
                                            default:
                                                System.out.println("Invalid operation");
                                                break;
                                        }
                                    }
                                }

    3. Month Days

  3. Write a Java program that takes an integer input (1-12) and prints the number of days in that month using a switch case. 
                            Sample Output
                        
                            Enter a month number (1-12): 2
                            28 or 29 days
                        
                    
                                
                                import java.util.Scanner;

                                public class MonthDays {
                                    public static void main(String[] args) {
                                        Scanner scanner = new Scanner(System.in);
                                        System.out.print("Enter a month number (1-12): ");
                                        int month = scanner.nextInt();
                                        switch (month) {
                                            case 1: case 3: case 5: case 7: case 8: case 10: case 12:
                                                System.out.println("31 days");
                                                break;
                                            case 4: case 6: case 9: case 11:
                                                System.out.println("30 days");
                                                break;
                                            case 2:
                                                System.out.println("28 or 29 days");
                                                break;
                                            default:
                                                System.out.println("Invalid input");
                                                break;
                                        }
                                    }
                                }

    4. Grade Evaluation

  4. Write a Java program that takes a character input representing a grade (A, B, C, D, F) and prints the corresponding description using a switch case. 
                            Sample Output
                        
                            Enter a grade (A, B, C, D, F): A
                            Excellent
                        
                    
                                
                                import java.util.Scanner;

                                public class GradeEvaluation {
                                    public static void main(String[] args) {
                                        Scanner scanner = new Scanner(System.in);
                                        System.out.print("Enter a grade (A, B, C, D, F): ");
                                        char grade = scanner.next().charAt(0);
                                        switch (grade) {
                                            case 'A':
                                                System.out.println("Excellent");
                                                break;
                                            case 'B':
                                                System.out.println("Good");
                                                break;
                                            case 'C':
                                                System.out.println("Average");
                                                break;
                                            case 'D':
                                                System.out.println("Below Average");
                                                break;
                                            case 'F':
                                                System.out.println("Fail");
                                                break;
                                            default:
                                                System.out.println("Invalid grade");
                                                break;
                                        }
                                    }
                                }

    5. Vowel or Consonant

  5. Write a Java program that takes a character input and checks whether it is a vowel or consonant using a switch case. 
                            Sample Output
                        
                            Enter a letter: a
                            The letter is a vowel.
                        
                    
                                
                                import java.util.Scanner;

                                public class VowelOrConsonant {
                                    public static void main(String[] args) {
                                        Scanner scanner = new Scanner(System.in);
                                        System.out.print("Enter a letter: ");
                                        char letter = scanner.next().charAt(0);
                                        switch (letter) {
                                            case 'a': case 'e': case 'i': case 'o': case 'u':
                                            case 'A': case 'E': case 'I': case 'O': case 'U':
                                                System.out.println("The letter is a vowel.");
                                                break;
                                            default:
                                                System.out.println("The letter is a consonant.");
                                                break;
                                        }
                                    }
                                }
