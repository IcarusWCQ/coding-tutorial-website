Check Palindrome

  1. Write a Python program to check if a given string is a palindrome. 
                            Sample Output
                        
                            Enter a string: madam
                            The string is a palindrome.
                        
                    
                                
                                def is_palindrome(s):
                                    return s == s[::-1]

                                s = input("Enter a string: ")
                                if is_palindrome(s):
                                    print("The string is a palindrome.")
                                else:
                                    print("The string is not a palindrome.")

    2. Fibonacci Series

  2. Write a Python program to generate the first n numbers in the Fibonacci series. 
                            Sample Output
                        
                            Enter the number of terms: 5
                            Fibonacci Series: 0 1 1 2 3
                        
                    
                                
                                def fibonacci(n):
                                    a, b = 0, 1
                                    for _ in range(n):
                                        yield a
                                        a, b = b, a + b

                                n = int(input("Enter the number of terms: "))
                                print("Fibonacci Series:", ' '.join(map(str, fibonacci(n))))

    3. Find the Largest Number in a List

  3. Write a Python program to find the largest number in a list. 
                            Sample Output
                        
                            List: [1, 2, 3, 4, 5]
                            The largest number is 5
                        
                    
                                
                                def find_largest(lst):
                                    return max(lst)

                                lst = [1, 2, 3, 4, 5]
                                print("List:", lst)
                                print("The largest number is", find_largest(lst))

    4. Sum of Digits in a Number

  4. Write a Python program to find the sum of digits in a number. 
                            Sample Output
                        
                            Enter a number: 123
                            Sum of digits: 6
                        
                    
                                
                                def sum_of_digits(n):
                                    return sum(int(digit) for digit in str(n))

                                n = int(input("Enter a number: "))
                                print("Sum of digits:", sum_of_digits(n))

    5. Count Vowels in a String

  5. Write a Python program to count the number of vowels in a given string. 
                            Sample Output
                        
                            Enter a string: hello world
                            Number of vowels: 3
                        
                    
                                
                                def count_vowels(s):
                                    vowels = 'aeiouAEIOU'
                                    return sum(1 for char in s if char in vowels)

                                s = input("Enter a string: ")
                                print("Number of vowels:", count_vowels(s))
