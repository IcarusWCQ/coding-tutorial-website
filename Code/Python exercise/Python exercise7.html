Fibonacci Sequence

  1. Write a Python function to generate the first n Fibonacci numbers. 
                            Sample Output
                        
                            Enter the number of terms: 5
                            Fibonacci Sequence: [0, 1, 1, 2, 3]
                        
                    
                                
                                def fibonacci(n):
                                    sequence = [0, 1]
                                    while len(sequence) < n:
                                        sequence.append(sequence[-1] + sequence[-2])
                                    return sequence[:n]

                                n = int(input("Enter the number of terms: "))
                                print("Fibonacci Sequence:", fibonacci(n))

    2. Prime Checker

  2. Write a Python function to check if a number is prime. 
                            Sample Output
                        
                            Enter a number: 29
                            29 is a prime number.
                        
                    
                                
                                def is_prime(n):
                                    if n <= 1:
                                        return False
                                    for i in range(2, int(n**0.5) + 1):
                                        if n % i == 0:
                                            return False
                                    return True

                                n = int(input("Enter a number: "))
                                if is_prime(n):
                                    print(f"{n} is a prime number.")
                                else:
                                    print(f"{n} is not a prime number.")

    3. Factorial Calculation

  3. Write a Python function to calculate the factorial of a number. 
                            Sample Output
                        
                            Enter a number: 5
                            Factorial of 5 is 120.
                        
                    
                                
                                def factorial(n):
                                    if n == 0:
                                        return 1
                                    else:
                                        return n * factorial(n-1)

                                n = int(input("Enter a number: "))
                                print(f"Factorial of {n} is {factorial(n)}.")

    4. Palindrome Checker

  4. Write a Python function to check if a string is a palindrome. 
                            Sample Output
                        
                            Enter a string: madam
                            madam is a palindrome.
                        
                    
                                
                                def is_palindrome(s):
                                    return s == s[::-1]

                                s = input("Enter a string: ")
                                if is_palindrome(s):
                                    print(f"{s} is a palindrome.")
                                else:
                                    print(f"{s} is not a palindrome.")

    5. Find the GCD

  5. Write a Python function to find the Greatest Common Divisor (GCD) of two numbers. 
                            Sample Output
                        
                            Enter first number: 56
                            Enter second number: 98
                            GCD of 56 and 98 is 14.
                        
                    
                                
                                def gcd(a, b):
                                    while b:
                                        a, b = b, a % b
                                    return a

                                a = int(input("Enter first number: "))
                                b = int(input("Enter second number: "))
                                print(f"GCD of {a} and {b} is {gcd(a, b)}.")
