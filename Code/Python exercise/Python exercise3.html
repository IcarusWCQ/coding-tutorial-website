If Statements

Check Even or Odd

  1. Write a Python program to check if a number is even or odd using an `if` statement. 
                            Sample Output
                        
                            Enter a number: 7
                            The number is odd.
                        
                    
                                
                                num = int(input("Enter a number: "))
                                if num % 2 == 0:
                                    print("The number is even.")
                                else:
                                    print("The number is odd.")

    2. Find the Largest Number

  2. Write a Python program to find the largest of three numbers using nested `if` statements. 
                            Sample Output
                        
                            Enter first number: 3
                            Enter second number: 7
                            Enter third number: 5
                            The largest number is 7.
                        
                    
                                
                                a = int(input("Enter first number: "))
                                b = int(input("Enter second number: "))
                                c = int(input("Enter third number: "))
                                
                                if a >= b and a >= c:
                                    largest = a
                                elif b >= a and b >= c:
                                    largest = b
                                else:
                                    largest = c

                                print("The largest number is", largest)

    3. Check Leap Year

  3. Write a Python program to check if a year is a leap year or not using an `if` statement. 
                            Sample Output
                        
                            Enter a year: 2020
                            The year is a leap year.
                        
                    
                                
                                year = int(input("Enter a year: "))
                                if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
                                    print("The year is a leap year.")
                                else:
                                    print("The year is not a leap year.")

    4. Grading System

  4. Write a Python program to assign a grade to a student based on their score using multiple `if` statements. 
                            Sample Output
                        
                            Enter the score: 85
                            The grade is: B
                        
                    
                                
                                score = int(input("Enter the score: "))

                                if score >= 90:
                                    grade = 'A'
                                elif score >= 80:
                                    grade = 'B'
                                elif score >= 70:
                                    grade = 'C'
                                elif score >= 60:
                                    grade = 'D'
                                else:
                                    grade = 'F'

                                print("The grade is:", grade)

    5. Check Number Sign

  5. Write a Python program to check if a number is positive, negative, or zero using an `if` statement. 
                            Sample Output
                        
                            Enter a number: -5
                            The number is negative.
                        
                    
                                
                                num = int(input("Enter a number: "))
                                
                                if num > 0:
                                    print("The number is positive.")
                                elif num == 0:
                                    print("The number is zero.")
                                else:
                                    print("The number is negative.")
