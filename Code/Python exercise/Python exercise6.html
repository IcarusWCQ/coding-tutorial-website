Arrays

Sum of Array Elements

  1. Write a Python program to calculate the sum of all elements in an array. 
                            Sample Output
                        
                            Array: [1, 2, 3, 4, 5]
                            Sum of all elements: 15
                        
                    
                                
                                arr = [1, 2, 3, 4, 5]
                                total = sum(arr)
                                print("Array:", arr)
                                print("Sum of all elements:", total)

    2. Find the Largest Element

  2. Write a Python program to find the largest element in an array. 
                            Sample Output
                        
                            Array: [1, 8, 3, 4, 5]
                            Largest element: 8
                        
                    
                                
                                arr = [1, 8, 3, 4, 5]
                                largest = max(arr)
                                print("Array:", arr)
                                print("Largest element:", largest)

    3. Reverse an Array

  3. Write a Python program to reverse an array. 
                            Sample Output
                        
                            Original array: [1, 2, 3, 4, 5]
                            Reversed array: [5, 4, 3, 2, 1]
                        
                    
                                
                                arr = [1, 2, 3, 4, 5]
                                reversed_arr = arr[::-1]
                                print("Original array:", arr)
                                print("Reversed array:", reversed_arr)

    4. Second Largest Element

  4. Write a Python program to find the second largest element in an array. 
                            Sample Output
                        
                            Array: [10, 20, 4, 45, 99]
                            Second largest element: 45
                        
                    
                                
                                arr = [10, 20, 4, 45, 99]
                                arr.sort()
                                second_largest = arr[-2]
                                print("Array:", arr)
                                print("Second largest element:", second_largest)

    5. Remove Duplicates

  5. Write a Python program to remove duplicates from an array. 
                            Sample Output
                        
                            Original array: [1, 2, 2, 3, 4, 4, 5]
                            Array after removing duplicates: [1, 2, 3, 4, 5]
                        
                    
                                
                                arr = [1, 2, 2, 3, 4, 4, 5]
                                arr_no_duplicates = list(set(arr))
                                arr_no_duplicates.sort()  # Sorting the list to maintain the order
                                print("Original array:", arr)
                                print("Array after removing duplicates:", arr_no_duplicates)
