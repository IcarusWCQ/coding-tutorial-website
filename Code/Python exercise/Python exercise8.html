Define a Class

  1. Write a Python program to create a class named `Rectangle` with attributes length and width, and a method to calculate the area. 
                            Sample Output
                        
                            Length: 5
                            Width: 3
                            Area: 15
                        
                    
                                
                                class Rectangle:
                                    def __init__(self, length, width):
                                        self.length = length
                                        self.width = width

                                    def area(self):
                                        return self.length * self.width

                                length = int(input("Length: "))
                                width = int(input("Width: "))
                                rect = Rectangle(length, width)
                                print("Area:", rect.area())

    2. Inheritance

  2. Write a Python program to create a base class `Person` with attributes name and age, and a derived class `Student` with an attribute grade. Include a method in the `Student` class to display the student's details. 
                            Sample Output
                        
                            Name: John
                            Age: 20
                            Grade: A
                            Student Details: Name: John, Age: 20, Grade: A
                        
                    
                                
                                class Person:
                                    def __init__(self, name, age):
                                        self.name = name
                                        self.age = age

                                class Student(Person):
                                    def __init__(self, name, age, grade):
                                        super().__init__(name, age)
                                        self.grade = grade

                                    def display(self):
                                        print(f"Student Details: Name: {self.name}, Age: {self.age}, Grade: {self.grade}")

                                name = input("Name: ")
                                age = int(input("Age: "))
                                grade = input("Grade: ")
                                student = Student(name, age, grade)
                                student.display()

    3. Method Overriding

  3. Write a Python program to create a base class `Shape` with a method `area` that returns 0. Create a derived class `Circle` with an attribute radius and override the `area` method to calculate the area of the circle. 
                            Sample Output
                        
                            Radius: 5
                            Area of Circle: 78.53981633974483
                        
                    
                                
                                import math

                                class Shape:
                                    def area(self):
                                        return 0

                                class Circle(Shape):
                                    def __init__(self, radius):
                                        self.radius = radius

                                    def area(self):
                                        return math.pi * self.radius ** 2

                                radius = float(input("Radius: "))
                                circle = Circle(radius)
                                print("Area of Circle:", circle.area())

    4. Encapsulation

  4. Write a Python program to create a class `BankAccount` with attributes account number and balance. Include methods to deposit and withdraw money, ensuring the balance never goes negative. 
                            Sample Output
                        
                            Account Number: 123456
                            Initial Balance: 1000
                            Deposit: 500
                            New Balance: 1500
                            Withdraw: 700
                            New Balance: 800
                            Withdraw: 900
                            Insufficient funds.
                            Balance: 800
                        
                    
                                
                                class BankAccount:
                                    def __init__(self, account_number, balance=0):
                                        self.account_number = account_number
                                        self.__balance = balance

                                    def deposit(self, amount):
                                        if amount > 0:
                                            self.__balance += amount
                                        else:
                                            print("Deposit amount must be positive.")

                                    def withdraw(self, amount):
                                        if 0 < amount <= self.__balance:
                                            self.__balance -= amount
                                        else:
                                            print("Insufficient funds.")

                                    def get_balance(self):
                                        return self.__balance

                                account_number = input("Account Number: ")
                                initial_balance = float(input("Initial Balance: "))
                                account = BankAccount(account_number, initial_balance)
                                
                                deposit_amount = float(input("Deposit: "))
                                account.deposit(deposit_amount)
                                print("New Balance:", account.get_balance())
                                
                                withdraw_amount = float(input("Withdraw: "))
                                account.withdraw(withdraw_amount)
                                print("New Balance:", account.get_balance())

                                withdraw_amount = float(input("Withdraw: "))
                                account.withdraw(withdraw_amount)
                                print("Balance:", account.get_balance())

    5. Polymorphism

  5. Write a Python program to create a base class `Animal` with a method `sound` that returns a generic sound. Create derived classes `Dog` and `Cat` that override the `sound` method to return "Bark" and "Meow" respectively. Create a function that takes an object of the `Animal` class and calls its `sound` method. 
                            Sample Output
                        
                            Dog sound: Bark
                            Cat sound: Meow
                        
                    
                                
                                class Animal:
                                    def sound(self):
                                        return "Some sound"

                                class Dog(Animal):
                                    def sound(self):
                                        return "Bark"

                                class Cat(Animal):
                                    def sound(self):
                                        return "Meow"

                                def animal_sound(animal):
                                    print(animal.sound())

                                dog = Dog()
                                cat = Cat()
                                print("Dog sound:", dog.sound())
                                print("Cat sound:", cat.sound())
