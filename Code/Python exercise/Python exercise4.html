For Loop

  1. Write a Python program to print the first 10 natural numbers using a `for` loop. 
                            Sample Output

  3. Write a Python program to calculate the factorial of a number using a `for` loop. 
                            Sample Output
                        
                            Enter a number: 5
                            Factorial of 5 is 120
                        
                    
                                
                                num = int(input("Enter a number: "))
                                factorial = 1
                                for i in range(1, num + 1):
                                    factorial *= i
                                print(f"Factorial of {num} is {factorial}")

    4. While Loop

  4. Write a Python program to print all the numbers from 1 to a given number using a `while` loop. 
                            Sample Output
                        
                            Enter a number: 5
                            1 2 3 4 5
                        
                    
                                
                                num = int(input("Enter a number: "))
                                i = 1
                                while i <= num:
                                    print(i, end=' ')
                                    i += 1
  5. Write a Python program to find the sum of all numbers from 1 to a given number using a `while` loop. 
                            Sample Output
                        
                            Enter a number: 10
                            Sum of numbers from 1 to 10 is 55
                        
                    
                                
                                num = int(input("Enter a number: "))
                                sum = 0
                                i = 1
                                while i <= num:
                                    sum += i
                                    i += 1
                                print(f"Sum of numbers from 1 to {num} is {sum}")
  6. Write a Python program to reverse a given number using a `while` loop. 
                            Sample Output
                        
                            Enter a number: 1234
                            Reversed number is 4321
                        
                    
                                
                                num = int(input("Enter a number: "))
                                reversed_num = 0
                                while num > 0:
                                    remainder = num % 10
                                    reversed_num = (reversed_num * 10) + remainder
                                    num //= 10
                                print(f"Reversed number is {reversed_num}")

    7. Do-While Loop (Simulated with While Loop)

  7. Write a Python program to print numbers from 1 to a given number using a `do-while` loop (simulated with a `while` loop). 
                            Sample Output
                        
                            Enter a number: 5
                            1 2 3 4 5
                        
                    
                                
                                num = int(input("Enter a number: "))
                                i = 1
                                while True:
                                    print(i, end=' ')
                                    i += 1
                                    if i > num:
                                        break
  8. Write a Python program to print the multiplication table of a given number using a `do-while` loop (simulated with a `while` loop).

                            3 x 5 = 15

                                i = 1
                                while True:
                                    print(f"{num} x {i} = {num * i}")

  9. Write a Python program to find the sum of digits of a number using a `do-while` loop (simulated with a `while` loop). 
                            Sample Output
                        
                            Enter a number: 123
                            Sum of digits: 6
                        
                    
                                
                                num = int(input("Enter a number: "))
                                sum_of_digits = 0
                                while True:
                                    sum_of_digits += num % 10
                                    num //= 10
                                    if num == 0:
                                        break
                                print(f"Sum of digits: {sum_of_digits}")
