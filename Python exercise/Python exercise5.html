Logical Operators

AND Operator

  1. Write a Python program to check if a number is between 10 and 20 (inclusive) using the `AND` operator. 
                            Sample Output
                        
                            Enter a number: 15
                            The number is between 10 and 20.
                        
                    
                                
                                num = int(input("Enter a number: "))
                                if num >= 10 and num <= 20:
                                    print("The number is between 10 and 20.")
                                else:
                                    print("The number is not between 10 and 20.")
  2. Write a Python program to check if a character is a vowel (a, e, i, o, u) using the `AND` operator. 
                            Sample Output
                        
                            Enter a character: e
                            The character is a vowel.
                        
                    
                                
                                char = input("Enter a character: ")
                                if char == 'a' or char == 'e' or char == 'i' or char == 'o' or char == 'u':
                                    print("The character is a vowel.")
                                else:
                                    print("The character is not a vowel.")
  3. Write a Python program to check if a person is eligible to vote (age >= 18 and is a citizen) using the `AND` operator. 
                            Sample Output
                        
                            Enter age: 20
                            Are you a citizen (yes/no): yes
                            You are eligible to vote.
                        
                    
                                
                                age = int(input("Enter age: "))
                                citizen = input("Are you a citizen (yes/no): ")
                                if age >= 18 and citizen == "yes":
                                    print("You are eligible to vote.")
                                else:
                                    print("You are not eligible to vote.")

    4. OR Operator

  4. Write a Python program to check if a number is less than 10 or greater than 20 using the `OR` operator. 
                            Sample Output
                        
                            Enter a number: 25
                            The number is less than 10 or greater than 20.
                        
                    
                                
                                num = int(input("Enter a number: "))
                                if num < 10 or num > 20:
                                    print("The number is less than 10 or greater than 20.")
                                else:
                                    print("The number is between 10 and 20.")
  5. Write a Python program to check if a character is a vowel (a, e, i, o, u) using the `OR` operator.

    7. NOT Operator

  7. Write a Python program to check if a number is not equal to 10 using the `NOT` operator. 
                            Sample Output
                        
                            Enter a number: 15
                            The number is not equal to 10.
                        
                    
                                
                                num = int(input("Enter a number: "))
                                if not num == 10:
                                    print("The number is not equal to 10.")
                                else:
                                    print("The number is equal to 10.")
  8. Write a Python program to check if a character is not a vowel (a, e, i, o, u) using the `NOT` operator. 
                            Sample Output
                        
                            Enter a character: k
                            The character is not a vowel.
                        
                    
                                
                                char = input("Enter a character: ")
                                if not (char == 'a' or char == 'e' or char == 'i' or char == 'o' or char == 'u'):
                                    print("The character is not a vowel.")
                                else:
                                    print("The character is a vowel.")
  9. Write a Python program to check if a person is not eligible to vote (age < 18) using the `NOT` operator. 
                            Sample Output
                        
                            Enter age: 16
                            You are not eligible to vote.
                        
                    
                                
                                age = int(input("Enter age: "))
                                if not age >= 18:
                                    print("You are not eligible to vote.")
                                else:
                                    print("You are eligible to vote.")
