Inheritance

Single Inheritance

  1. Write a Python program to create a base class `Animal` with a method `sound` that prints a generic sound. Create a derived class `Dog` that overrides the `sound` method to print "Bark". 
                            Sample Output
                        
                            Animal sound:
                            Some generic sound
                            Dog sound:
                            Bark
                        
                    
                                
                                class Animal:
                                    def sound(self):
                                        print("Some generic sound")

                                class Dog(Animal):
                                    def sound(self):
                                        print("Bark")

                                animal = Animal()
                                dog = Dog()
                                print("Animal sound:")
                                animal.sound()
                                print("Dog sound:")
                                dog.sound()

    2. Multiple Inheritance

  2. Write a Python program to create two base classes `Person` and `Employee`. The `Person` class should have attributes `name` and `age`, and the `Employee` class should have an attribute `employee_id`. Create a derived class `Manager` that inherits from both `Person` and `Employee`, and includes an attribute `department`. 
                            Sample Output
                        
                            Name: Alice
                            Age: 35
                            Employee ID: E123
                            Department: HR
                        
                    
                                
                                class Person:
                                    def __init__(self, name, age):
                                        self.name = name
                                        self.age = age

                                class Employee:
                                    def __init__(self, employee_id):
                                        self.employee_id = employee_id

                                class Manager(Person, Employee):
                                    def __init__(self, name, age, employee_id, department):
                                        Person.__init__(self, name, age)
                                        Employee.__init__(self, employee_id)
                                        self.department = department

                                    def display(self):
                                        print(f"Name: {self.name}")
                                        print(f"Age: {self.age}")
                                        print(f"Employee ID: {self.employee_id}")
                                        print(f"Department: {self.department}")

                                manager = Manager("Alice", 35, "E123", "HR")
                                manager.display()

    3. Multilevel Inheritance

  3. Write a Python program to create a base class `Vehicle` with an attribute `make`. Create a derived class `Car` that inherits from `Vehicle` and includes an attribute `model`. Further, create a derived class `ElectricCar` that inherits from `Car` and includes an attribute `battery_capacity`. 
                            Sample Output
                        
                            Make: Tesla
                            Model: Model S
                            Battery Capacity: 100 kWh
                        
                    
                                
                                class Vehicle:
                                    def __init__(self, make):
                                        self.make = make

                                class Car(Vehicle):
                                    def __init__(self, make, model):
                                        super().__init__(make)
                                        self.model = model

                                class ElectricCar(Car):
                                    def __init__(self, make, model, battery_capacity):
                                        super().__init__(make, model)
                                        self.battery_capacity = battery_capacity

                                    def display(self):
                                        print(f"Make: {self.make}")
                                        print(f"Model: {self.model}")
                                        print(f"Battery Capacity: {self.battery_capacity} kWh")

                                electric_car = ElectricCar("Tesla", "Model S", 100)
                                electric_car.display()

    4. Hierarchical Inheritance

  4. Write a Python program to create a base class `Shape` with an attribute `name`. Create two derived classes `Circle` and `Square` that inherit from `Shape` and include methods to calculate the area. 
                            Sample Output
                        
                            Circle area: 78.53981633974483
                            Square area: 25
                        
                    
                                
                                import math

                                class Shape:
                                    def __init__(self, name):
                                        self.name = name

                                class Circle(Shape):
                                    def __init__(self, name, radius):
                                        super().__init__(name)
                                        self.radius = radius

                                    def area(self):
                                        return math.pi * self.radius ** 2

                                class Square(Shape):
                                    def __init__(self, name, side_length):
                                        super().__init__(name)
                                        self.side_length = side_length

                                    def area(self):
                                        return self.side_length ** 2

                                circle = Circle("Circle", 5)
                                square = Square("Square", 5)
                                print(f"{circle.name} area:", circle.area())
                                print(f"{square.name} area:", square.area())

    5. Hybrid Inheritance

  5. Write a Python program to create a base class `Person` with an attribute `name`. Create two derived classes `Student` and `Teacher` that inherit from `Person` and include attributes `student_id` and `teacher_id` respectively. Further, create a derived class `TeachingAssistant` that inherits from both `Student` and `Teacher`, and includes an attribute `hours_per_week`. 
                            Sample Output
                        
                            Name: Bob
                            Student ID: S123
                            Teacher ID: T456
                            Hours per week: 20
                        
                    
                                
                                class Person:
                                    def __init__(self, name):
                                        self.name = name

                                class Student(Person):
                                    def __init__(self, name, student_id):
                                        super().__init__(name)
                                        self.student_id = student_id

                                class Teacher(Person):
                                    def __init__(self, name, teacher_id):
                                        super().__init__(name)
                                        self.teacher_id = teacher_id

                                class TeachingAssistant(Student, Teacher):
                                    def __init__(self, name, student_id, teacher_id, hours_per_week):
                                        Student.__init__(self, name, student_id)
                                        Teacher.__init__(self, name, teacher_id)
                                        self.hours_per_week = hours_per_week

                                    def display(self):
                                        print(f"Name: {self.name}")
                                        print(f"Student ID: {self.student_id}")
                                        print(f"Teacher ID: {self.teacher_id}")
                                        print(f"Hours per week: {self.hours_per_week}")

                                ta = TeachingAssistant("Bob", "S123", "T456", 20)
                                ta.display()
