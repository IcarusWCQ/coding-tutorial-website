Break and Continue

Break

  1. Write a Java program to print numbers from 1 to 10. If the number is 5, break the loop. 
                            Sample Output
                        
1 2 3 4
  2. Write a Java program to find the first multiple of 7 in an array and break the loop. 
                            Sample Output
                        
The first multiple of 7 is 14
  3. Write a Java program to check if a number is prime. If a divisor is found, break the loop. 
                            Sample Output
                        
The number is not prime

Continue

  1. Write a Java program to print numbers from 1 to 10. If the number is 5, skip it using continue. 
                            Sample Output
                        
1 2 3 4 6 7 8 9 10
  2. Write a Java program to print the first 10 multiples of 3, but skip multiples of 4 using continue. 
                            Sample Output
                        
3 9 15 21 27
  3. Write a Java program to print numbers from 1 to 10, but skip even numbers using continue. 
                            Sample Output
                        
1 3 5 7 9
