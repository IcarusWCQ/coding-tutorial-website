Logical Operators

AND Operator

  1. Write a Java program to check if a number is between 10 and 20 (inclusive) using the AND operator. 
                            Sample Output
                        
The number is between 10 and 20.
  2. Write a Java program to check if a character is a vowel (a, e, i, o, u) using the AND operator. 
                            Sample Output
                        
The character is a vowel.
  3. Write a Java program to check if a person is eligible to vote (age >= 18 and is a citizen) using the AND operator. 
                            Sample Output
                        
You are eligible to vote.

OR Operator

  1. Write a Java program to check if a number is less than 10 or greater than 20 using the OR operator. 
                            Sample Output
                        
The number is less than 10 or greater than 20.
  2. Write a Java program to check if a character is a vowel (a, e, i, o, u) using the OR operator. 
                            Sample Output
                        
The character is a vowel.
  3. Write a Java program to check if a person is eligible for a discount (age < 18 or age > 60) using the OR operator. 
                            Sample Output
                        
You are eligible for a discount.

NOT Operator

  1. Write a Java program to check if a number is not equal to 10 using the NOT operator. 
                            Sample Output
                        
The number is not equal to 10.
  2. Write a Java program to check if a character is not a vowel (a, e, i, o, u) using the NOT operator. 
                            Sample Output
                        
The character is not a vowel.
  3. Write a Java program to check if a person is not eligible to vote (age < 18) using the NOT operator. 
                            Sample Output
                        
You are not eligible to vote.
